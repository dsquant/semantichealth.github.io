# Data

The directories under this one contain the code and infrastructure components used to create the Semantic Health site.

+ **Data Ingest** contains the code and processes by which data is taken in and processed into the Elasticsearch index that drives the site.

+ **Elasticsearch** contains the infrastructure code used to build the Elasticsearch cluster in IBM SoftLayer.

+ **MongoDB** contains the infrastructure code used to build the MongoDB server in AWS EC2.

+ **PostgreSQL** contains the DDL for the PostgreSQL database used to track data ingest processing.

+ **Scrape** contains the code used to scrape the web and other data sources.
